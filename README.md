# Random Hello + Reply

A tiny, single-file website that displays a random “Hello” greeting and a friendly reply. It shows a new random pair on page load and whenever you click the button.

How to use
- Open index.html in any modern browser.
- Click “New random pair” or press the R key to see a new greeting and reply.

What’s included
- index.html — everything (HTML, CSS, JS) in one file.

Notes
- No build step or dependencies.
- Works fully offline.
